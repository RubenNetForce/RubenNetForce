\# Secure ELK Stack Installer for Linux with Docker



This repository contains a script to \*\*install a secure ELK stack\*\* (Elasticsearch, Kibana, Logstash) on Ubuntu or any Linux distribution using Docker.  

The installation is \*\*fully automated\*\* and requires minimal configuration.



---



\##  Important



The script generates a \*\*`.env` file\*\* containing your credentials (Elastic and Kibana passwords).  

Do \*\*not\*\* share this file after execution, as it contains sensitive information.



---



\##  Requirements



\- Docker installed

\- Docker Compose installed (version 3.8+)

\- Minimum 2 GB RAM recommended

\- Linux server (Ubuntu, Debian, CentOS, etc.)
